Raspberry Pi NAS (OMV)

A complete, dedicated NAS solution with a web GUI, supporting SMB/CIFS, (S)FTP, and RSync out of the box.

Overview

OpenMediaVault (OMV) is a next-generation network attached storage (NAS) solution based on Debian Linux. It contains services like SSH, (S)FTP, SMB/CIFS, DAAP media server, RSync, BitTorrent client and many more. It's an out-of-the-box solution that transforms your Raspberry Pi into a professional-grade storage server.

💡 Note: OMV dedicates your Pi to NAS duty; running other complex software alongside it can break the installation.

Terminal Setup

  • 1. Update the System

    sudo apt update && sudo apt upgrade -y

  • 2. Run the Pre-Install Script

    Sets up a persistent Ethernet connection to avoid connectivity issues:

    wget -O - https://raw.githubusercontent.com/OpenMediaVault-Plugin-Developers/installScript/master/preinstall | sudo bash

  • 3. Install OMV

    This installs everything needed. Takes several minutes.

    wget -O - https://raw.githubusercontent.com/OpenMediaVault-Plugin-Developers/installScript/master/install | sudo bash

    • Web UI Configuration

    Access the interface at http://<YOUR_PI_IP> (Default: admin / openmediavault).

  • Storage Setup

    Go to Storage > Disks to verify your drive, then File Systems to create an EXT4 partition and mount it.

  • Sharing

    Create a Shared Folder, then enable the SMB/CIFS service to make the drive visible on your Windows/Mac network.

    • ⚠ Security: Change the default admin password immediately after your first login.
