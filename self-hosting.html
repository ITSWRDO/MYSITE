Self-Hosting on Raspberry Pi 5

A zero-cost, no-port-forwarding web hosting setup running raunakkushwaha.tech from home.

Overview

I set up my personal website raunakkushwaha.tech running entirely from a Raspberry Pi 5 at home — no port forwarding, no static IP, no paid hosting. Cloudflare Tunnel creates a secure outbound connection, keeping my home IP hidden while serving the site globally over HTTPS.

Stack

Hardware Raspberry Pi 5
OS Pi OS Lite
Web Server Nginx
Tunnel Cloudflare
Domain get.tech
SSL Auto HTTPS

Quick Setup

One-liner
curl -fsSL https://raw.githubusercontent.com/ITSWRDO/pi-setup/main/setup.sh | bash
⚠ Security: Always inspect scripts before running them with elevated privileges.

Commands

Tunnel Management
sudo systemctl restart cloudflared cloudflared tunnel info raunak-pi
Nginx
sudo nginx -t sudo systemctl reload nginx
Permissions Fix
sudo chmod 755 /home/wrdo/ sudo chown -R www-data:www-data /home/wrdo/123/
